## 🏛️ About

**Libertad VZLA** is a high-performance journalism platform engineered to deliver independent news from Venezuela. In a context where press freedom faces systematic suppression, this platform provides secure, resilient digital infrastructure for independent journalism.

### What We Do

- 📰 **Independent News Distribution** — Fast, accessible content delivery to Venezuelan citizens and the diaspora
- 🔒 **Source Protection** — Privacy-by-design architecture that protects journalist-source confidentiality
- ⚡ **Performance-First** — Core Web Vitals optimized with SSR, React Server Components, and edge deployment
- 🛡️ **Security-Hardened** — Built to resist state-level threats with defense-in-depth security

## 🔒 Security Commitment

This platform handles sensitive data in a high-risk geopolitical context. Our security posture includes:

- **Row-Level Security (RLS)** enforced on all database operations
- **Role-based access control** for editorial and administrative functions
- **Data minimization** — we collect only what is strictly necessary
- **Zero hardcoded credentials** — environment-based secrets management
- **Automated dependency auditing** and continuous security monitoring
- **Incident response protocols** with defined SLAs

> We take source protection and citizen data privacy as architectural requirements, not optional features.
